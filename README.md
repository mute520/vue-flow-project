# Vue 3 + Vite

This template should help get you started developing with Vue 3 in Vite. The template uses Vue 3 `<script setup>` SFCs, check out the [script setup docs](https://v3.vuejs.org/api/sfc-script-setup.html#sfc-script-setup) to learn more.

Learn more about IDE Support for Vue in the [Vue Docs Scaling up Guide](https://vuejs.org/guide/scaling-up/tooling.html#ide-support).

### 开始节点
工作流的起始节点，用于设定启动工作流需要的信息

### 插件节点
通过添加工具访问实时数据和执行外部操作

<!-- 当你需要获取网页、pdf、doc、docx、xlsx、csv、text 内容时，使用此工具，可以获取url链接下的标题和内容。由于个别网站自身站点限制，无法获取网页内容。 -->

### 大模型
调用大语言模型,使用变量和提示词生成回复

### 代码节点
编写代码，处理输入变量来生成返回值

### 条件节点
根据输入变量判断条件是否成立，若成立则运行对应的分支，若均不成立则只运行“否则”分支

### 循环节点
用于通过设定循环次数和逻辑，重复执行一系列任务

### 意图节点
用于用户输入的意图识别，并将其与预设意图选项进行匹配。

### 批处理节点
通过设定批量运行次数和逻辑，运行批处理体内的任务

### 变量
对多个分支的输出进行聚合处理

### 输出
节点从“消息”更名为“输出”，支持中间过程的消息输出，支持流式和非流式两种方式

### 图像生成
通过文字描述/添加参考图生成图片

### 工作流
集成已发布工作流，可以执行嵌套子任务

### 知识库检索
在选定的知识中,根据输入变量召回最匹配的信息,并以列表形式返回

### 图像生成
通过文字描述/添加参考图生成图片

### 异步任务
触发在任务中心预定义的异步任务，创建新的任务运行实例。不阻塞主工作流运行，可在任务中心查看对应的实例状态
